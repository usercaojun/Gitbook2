# VS Code 开启对 Markdown 实时预览

----------------------------------

在VSCode中安装插件可以实现对.md文件的实时预览

![cmd-markdown-logo](images/yulan.png)

**方法1：markdown**

1.用 VS code 打开 .md 文件  
2.按 F1 输入 markdown 选择下图的选项即可 

![cmd-markdown-logo](images/youceyulan.png)

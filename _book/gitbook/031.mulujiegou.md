# 目录结构

----------------------------------

**·**  
├─ book.json  
├─ README.md  
├─ SUMMARY.md  
├─ GLOSSARY.md  
├─ _book/
├─ .gitignore、.bookignore、.ignore过滤文件或文件夹

  
<table>
    <tr>
        <td>book.json</td>
        <td>配置数据 (可选)，但是作者认为必须得有，个人习惯</td>
    </tr>
    <tr>
        <td>README.md</td>
        <td>书本的第一页内容，通常是目录或封面</td>
    </tr>
    <tr>
        <td>SUMMARY.md</td>
        <td>用来定义章节或子章节目录结构，并用于生成图书目录</td>
    </tr>
    <tr>
        <td>GLOSSARY.md</td>
        <td>词汇/注释术语列表 (可选)</td>
    </tr>
</table>
